## Automatic Speech Recognition using NLP 

This project explores speech recognition and natural language processing (NLP) using the LibriSpeech dataset. It focuses on analyzing both audio signals and their text transcripts, demonstrating how raw speech data can be transformed into meaningful transcriptions.



### Techniques Used

- Audio Processing
- Mel-Frequency Cepstral Coefficients (MFCCs)
- Signal preprocessing and feature extraction
- Text Processing
- Text cleaning and normalization
- Bag-of-Words (BoW)
- TF-IDF vectorization


### Tech Stack

- Python
- Jupyter Notebook
- Libraries:
    - librosa
    - numpy
    - pandas
    - scikit-learn
    - matplotlib
